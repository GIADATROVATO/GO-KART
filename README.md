# GO-KART
Questo progetto consiste in un sistema di gestione per gare di Go Kart, progettato per modellare e coordinare sessioni di gara, piloti, kart ed eventi correlati.

L’obiettivo principale non era solo implementare funzionalità, ma comprendere quando e perché applicare un Design Pattern, valutandone l’impatto sulla fluidità del codice.

Il progetto è stato sviluppato con finalità formative e di approfondimento architetturale, con particolare attenzione alla progettazione orientata agli oggetti e all’applicazione consapevole dei Design Pattern.
